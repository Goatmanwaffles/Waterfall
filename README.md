# DatabaseFinalProject

# Running
### XAMPP
First, run Xampp, we have seed data in the sql folder
### Packages
##### For pip
```
python3 -m venv venv
./venv/bin/activate
pip install -r requirements.txt
```
##### For uv
```
uv sync
uv run app.py
```

### Testing
Some available testing accounts exist with logins
```
admin <- Username
abc   <- Password

instructor
abc

abc
```

More accounts are available in the seed_data.sql file, all with password abc for testing
### Procedures
To run the basic CRUD test procedures use
```
uv run test_procedures.py
```

OR

```
python3 test_procedures.py
```

# DEVELOPMENT
RUN 
```
docker compose up --build
```
This builds the docker container and runs all nessecarry commands. Opens port 4500 to run the local app
# File Structure
The sql folder contains the database schema as well as the test data generated by running the above app. It also contains our test queries for basic CRUD operation.

setup.py contains the logic for generating our test data.
